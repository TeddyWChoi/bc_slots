## ❓ 심볼 설계 규칙

| 항목 | 내용 |
|------|------|
| **트리거 조건** | 3개 릴 중 1개 이상에 ❓ 등장 시 발동 |
| **효과** | 무료 스핀 1회 추가 (베팅 차감 없음) |
| **가중치** | 5,000 / 100,000 |
| **출현 확률 (결과 기준)** | 5.000% |
| **릴 1개당 환산 확률** | ≈ 1.7% (P(at least 1 of 3 reels) = 1 − 0.983³ ≈ 5%) |
| **RTP 기여** | 5.000% × RTP(83%) = **4.15%** |
| **등급** | BONUS |

### 설계 근거

- 일반 슬롯머신 프리 스핀 트리거 표준 범위: **3 ~ 8% / 스핀**
- 균형형(Balanced) RTP 83% 설계 기준 → **5%** 채택
- 기존 가중치를 일괄 **×0.95** 축소 후 ? 에 5,000 가중치 신규 배분
- 프리 스핀 1회의 기대값 = 게임 RTP = **×0.83** (현금 배율 환산)
- 전체 RTP 변동: 83.0% → **83.3%** (허용 오차 범위 내)
